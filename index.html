HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The header element appears at the top of the page that contains a navigation menu and/or introductory content
  • The footer element appears at the bottom of the page that typically contains information about the author, navigation, contact, sitemap, etc.

CSS

  • Margin allows you to decide how much space you want around outside of an element
  • Padding allows you to decide how much space you want inside an element
  • The card elements designates any HTML element under the class card to have the property assignments from the CSS

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds code or information to GitHub.
  • git commit -m "example description": creates a commit

JavaScript

  • Variables are a containter that is used to store specific data
  • Arrays are single variables that can hold a group of data
  • Control flow is the order in which a computer executes code in a script
  • To execute a function we must call it in our code
  • When creating a function, we are defining a part of the code that performs a certain task and giving it a specific name
